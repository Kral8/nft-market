# Royal NFT Market - Telegram Mini App

Full-featured NFT marketplace on TON blockchain, designed as a Telegram Mini App.

## Features
- Mint and list NFTs directly in Telegram
- Buy NFTs with TON cryptocurrency
- IPFS storage via Pinata
- Firebase Firestore database
- TON Connect 2.0 wallet integration

## Setup
1. Clone repository
2. Configure Firebase project
3. Set up Pinata account
4. Deploy to GitHub Pages
5. Connect to Telegram Bot via @BotFather

## Security Notes
- Never commit API keys to repository
- Use Telegram CloudStorage for sensitive data
- Validate all transactions on backend
