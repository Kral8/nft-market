Pinata JWT Setup

How to get Pinata JWT:

  1. Go to pinata.cloud
  2. Create account or login
  3. Go to API Keys section
  4. Click "New API Key"
  5. Select permissions: pinFileToIPFS
  6. Copy the JWT token
  7. Paste it in the field above
